# Java Translation Notes

Use this folder for the Java version of the expense tracker.

Suggested translation steps:

1. Create an `Expense` class with fields for amount, category, description, and date.
2. Create an `ExpenseTracker` class that stores an `ArrayList<Expense>`.
3. Move menu actions into methods like `addExpense`, `viewExpenses`, and `showTotal`.
4. Use `Scanner` for console input.
5. Use `BufferedReader` and `BufferedWriter` for loading and saving CSV data.

Try translating one Python function at a time instead of rewriting everything all at once.
